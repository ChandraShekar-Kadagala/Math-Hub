First Order Differential Equations

Learn to identify the type and apply the correct method. Every type has its own pattern — master the pattern, master the marks.

Method 1
➗ Variable Separable Equations
Form: \(\dfrac{dy}{dx} = g(x) \cdot h(y)\)

Method: Rearrange to \(\dfrac{1}{h(y)}\,dy = g(x)\,dx\), then integrate both sides.
⚠️ Don't forget to add +C after integration! And check if \(h(y)=0\) gives a constant solution.
Solved — Q2 of Assignment

Solve: \(\dfrac{dy}{dx} = \dfrac{y^2+1}{x^2+1}\)

  1. Separate: \(\dfrac{dy}{y^2+1} = \dfrac{dx}{x^2+1}\)
  2. Integrate both sides: \(\displaystyle\int\frac{dy}{y^2+1} = \int\frac{dx}{x^2+1}\)
  3. \(\arctan(y) = \arctan(x) + C\)
Answer: \(\tan^{-1}(y) - \tan^{-1}(x) = C\)
Method 2
📏 Linear First-Order Equations (Integrating Factor)
Standard Form: \(\dfrac{dy}{dx} + P(x)y = Q(x)\)

Integrating Factor: \(IF = e^{\int P(x)\,dx}\)

Solution: \(y \cdot IF = \displaystyle\int Q(x) \cdot IF\,dx + C\)
Solved — Q1 of Assignment

Solve: \(\dfrac{dy}{dx} + 2y = e^{-x}\)

  1. Identify: \(P(x) = 2\), \(Q(x) = e^{-x}\)
  2. Compute IF: \(IF = e^{\int 2\,dx} = e^{2x}\)
  3. Multiply both sides by \(e^{2x}\): \(\dfrac{d}{dx}[y \cdot e^{2x}] = e^{-x} \cdot e^{2x} = e^x\)
  4. Integrate: \(y e^{2x} = \int e^x\,dx = e^x + C\)
  5. Solve for \(y\): \(y = e^{-x} + Ce^{-2x}\)
Answer: \(y = e^{-x} + Ce^{-2x}\)
Method 3
✅ Exact Differential Equations
Form: \(M(x,y)\,dx + N(x,y)\,dy = 0\)

Test for Exactness: \(\dfrac{\partial M}{\partial y} = \dfrac{\partial N}{\partial x}\)

Solution: Find \(\phi(x,y)\) such that \(\dfrac{\partial\phi}{\partial x} = M\) and \(\dfrac{\partial\phi}{\partial y} = N\). Then \(\phi(x,y) = C\).
  1. Test: compute \(\partial M/\partial y\) and \(\partial N/\partial x\). If equal → exact.
  2. Integrate \(M\) with respect to \(x\): \(\phi = \int M\,dx + g(y)\)
  3. Differentiate \(\phi\) w.r.t. \(y\), set equal to \(N\), solve for \(g'(y)\)
  4. Integrate to get \(g(y)\). Write final solution \(\phi(x,y)=C\).
Solved — Q5 of Assignment

Solve: \((2xy+y^2)\,dx + (x^2+2xy)\,dy = 0\)

  1. \(M = 2xy+y^2\), \(N = x^2+2xy\). Check: \(\partial M/\partial y = 2x+2y = \partial N/\partial x\). ✓ Exact.
  2. \(\phi = \int(2xy+y^2)\,dx = x^2y + xy^2 + g(y)\)
  3. \(\partial\phi/\partial y = x^2+2xy + g'(y) = N = x^2+2xy \Rightarrow g'(y)=0 \Rightarrow g(y)=C\)
Answer: \(x^2y + xy^2 = C\)
Method 4
🔧 Bernoulli Equation — Reduction to Linear
Form: \(\dfrac{dy}{dx} + P(x)y = Q(x)y^n\)   (\(n \neq 0, 1\))

Substitution: Let \(v = y^{1-n}\), then \(\dfrac{dv}{dx} = (1-n)y^{-n}\dfrac{dy}{dx}\)

This transforms into a linear equation in \(v\).
Solved — Q3 of Assignment

Solve: \(\dfrac{dy}{dx} + \dfrac{2}{x}y = x^3 y^3\)   (\(n=3\))

  1. Divide by \(y^3\): \(y^{-3}\dfrac{dy}{dx} + \dfrac{2}{x}y^{-2} = x^3\)
  2. Let \(v = y^{-2} \Rightarrow \dfrac{dv}{dx} = -2y^{-3}\dfrac{dy}{dx}\). So \(-\dfrac{1}{2}\dfrac{dv}{dx} + \dfrac{2}{x}v = x^3\)
  3. Standard form: \(\dfrac{dv}{dx} - \dfrac{4}{x}v = -2x^3\)
  4. IF = \(e^{\int -4/x\,dx} = x^{-4}\). Multiply: \(\dfrac{d}{dx}[vx^{-4}] = -2x^{-1}\)
  5. Integrate: \(vx^{-4} = -2\ln|x| + C \Rightarrow y^{-2} = x^4(C-2\ln|x|)\)
Answer: \(y^{-2} = x^4(C - 2\ln|x|)\)
Method 5
🔁 Homogeneous First-Order Equations
Form: \(\dfrac{dy}{dx} = f\!\left(\dfrac{y}{x}\right)\)

Substitution: Let \(y = vx \Rightarrow \dfrac{dy}{dx} = v + x\dfrac{dv}{dx}\)

Equation becomes separable in \(v\) and \(x\).
Solved — Q8 of Assignment

Solve: \(\dfrac{dy}{dx} = \dfrac{x+y}{x-y}\)

  1. Let \(y=vx\): \(v + x\dfrac{dv}{dx} = \dfrac{1+v}{1-v}\)
  2. \(x\dfrac{dv}{dx} = \dfrac{1+v}{1-v} - v = \dfrac{1+v^2}{1-v}\)
  3. Separate: \(\dfrac{1-v}{1+v^2}\,dv = \dfrac{dx}{x}\)
  4. \(\int\dfrac{1}{1+v^2}\,dv - \int\dfrac{v}{1+v^2}\,dv = \int\dfrac{dx}{x}\)
  5. \(\arctan(v) - \tfrac{1}{2}\ln(1+v^2) = \ln|x| + C\)
  6. Back-substitute \(v = y/x\): \(\arctan(y/x) - \tfrac{1}{2}\ln\!\left(\dfrac{x^2+y^2}{x^2}\right) = \ln|x| + C\)
Answer: \(\tan^{-1}(y/x) - \ln\sqrt{x^2+y^2} = C\)
Initial Value Problems (IVP) — Solved Q4

Solve: \(\dfrac{dy}{dx} + y\tan x = y^2\sec x\),   \(y(0) = 1\)

  1. Bernoulli with \(n=2\). Let \(v = y^{-1}\). Equation becomes: \(\dfrac{dv}{dx} - v\tan x = -\sec x\)
  2. IF = \(e^{\int -\tan x\,dx} = \cos x\). Multiply through: \(\dfrac{d}{dx}[v\cos x] = -\sec x\cos x = -1\)
  3. Integrate: \(v\cos x = -x + C\). So \(\dfrac{\cos x}{y} = C - x\).
  4. Apply IC \(y(0)=1\): \(\dfrac{1}{1} = C - 0 \Rightarrow C = 1\).
Answer: \(y = \dfrac{\cos x}{1-x}\)
Theory
📖 Existence & Uniqueness Theorem (Picard's)

For the IVP: \(\dfrac{dy}{dx} = f(x,y)\), \(y(x_0)=y_0\)

Existence

If \(f(x,y)\) is continuous in a rectangle around \((x_0,y_0)\), then at least one solution exists.

Uniqueness

If additionally \(\dfrac{\partial f}{\partial y}\) is continuous, then the solution is unique.
