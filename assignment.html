Differential Equations Solutions

Fully answered, step-by-step solutions for all 12 questions from the ODE Assignment (Unit 3 & Unit 4).

Q1. Solve: $\frac{dy}{dx} + 2y = e^{-x}$

Linear ODE

This is a linear first-order differential equation of the form $\frac{dy}{dx} + P(x)y = Q(x)$, where $P(x) = 2$ and $Q(x) = e^{-x}$.

Step 1: Integrating Factor (IF) = $e^{\int P(x) dx} = e^{\int 2 dx} = e^{2x}$.

Step 2: General Solution: $y \cdot (\text{IF}) = \int Q(x) \cdot (\text{IF}) dx + C$

$y e^{2x} = \int e^{-x} \cdot e^{2x} dx = \int e^x dx$

$y e^{2x} = e^x + C$

✅ Final Answer: $y = e^{-x} + C e^{-2x}$

Q2. Solve: $\frac{dy}{dx} = \frac{y^2+1}{x^2+1}$

Separable ODE

Using the method of Separation of Variables:

Step 1: $\frac{dy}{y^2+1} = \frac{dx}{x^2+1}$

Step 2: Integrating both sides: $\int \frac{1}{y^2+1} dy = \int \frac{1}{x^2+1} dx$

$\arctan(y) = \arctan(x) + C$

✅ Final Answer: $\arctan(y) - \arctan(x) = C$

Q3. Solve: $\frac{dy}{dx} + \frac{2}{x}y = x^3 y^3$

Bernoulli Equation

This is a Bernoulli's Equation of the form $\frac{dy}{dx} + P(x)y = Q(x)y^n$ with $n = 3$.

Step 1: Divide by $y^3$: $y^{-3} \frac{dy}{dx} + \frac{2}{x}y^{-2} = x^3$.

Step 2: Let $v = y^{-2}$. Then $\frac{dv}{dx} = -2y^{-3} \frac{dy}{dx}$.

Step 3: Substituting: $-\frac{1}{2} \frac{dv}{dx} + \frac{2}{x}v = x^3 \implies \frac{dv}{dx} - \frac{4}{x}v = -2x^3$.

Step 4: IF = $e^{\int -4/x dx} = x^{-4}$.

Step 5: $v \cdot x^{-4} = \int -2x^3 \cdot x^{-4} dx = \int -\frac{2}{x} dx = -2\ln|x| + C$.

✅ Final Answer: $y^{-2} = x^4(C - 2\ln|x|)$

Q4. Solve: $\frac{dy}{dx} + y\tan x = y^2 \sec x$ given $y(0) = 1$

Bernoulli Equation

Bernoulli equation ($n = 2$). Let $v = y^{-1}$.

Step 1: Divide by $y^2$: $y^{-2} \frac{dy}{dx} + y^{-1}\tan x = \sec x$.

Step 2: Substitute: $-\frac{dv}{dx} + v\tan x = \sec x \implies \frac{dv}{dx} - v\tan x = -\sec x$.

Step 3: IF = $e^{\int -\tan x dx} = \cos x$.

Step 4: $v \cos x = \int -\sec x \cos x dx = \int -1 dx = -x + C$.

Step 5: $\frac{\cos x}{y} = C - x$. Apply $y(0) = 1$: $\frac{1}{1} = C - 0 \implies C = 1$.

✅ Final Answer: $y = \frac{\cos x}{1 - x}$

Q5. Solve: $(2xy + y^2)dx + (x^2 + 2xy)dy = 0$

Exact ODE

Check for Exactness: $M = 2xy + y^2$, $N = x^2 + 2xy$.

Step 1: $\frac{\partial M}{\partial y} = 2x + 2y$, $\frac{\partial N}{\partial x} = 2x + 2y$. Since equal, it is exact.

Step 2: Integrate $M$ w.r.t $x$: $\int (2xy + y^2) dx = x^2y + xy^2 + g(y)$.

Step 3: Differentiate w.r.t $y$: $x^2 + 2xy + g'(y) = N = x^2 + 2xy$. Thus $g'(y) = 0 \implies g(y) = C$.

✅ Final Answer: $x^2y + xy^2 = C$

Q6. Solve: $(y - x)dx + (y + x)dy = 0$

Exact ODE

$M = y - x$, $N = y + x$. $\frac{\partial M}{\partial y} = 1$, $\frac{\partial N}{\partial x} = 1$. Exact.

Step 1: $\int M dx = \int (y - x) dx = xy - \frac{x^2}{2} + g(y)$.

Step 2: $\frac{\partial}{\partial y}(xy - \frac{x^2}{2} + g(y)) = x + g'(y) = y + x \implies g'(y) = y \implies g(y) = \frac{y^2}{2}$.

✅ Final Answer: $2xy - x^2 + y^2 = C$

Q7. Solve: $(2xy + y)dx + (x^2 + x)dy = 0$

Exact ODE

$M = 2xy + y$, $N = x^2 + x$. $\frac{\partial M}{\partial y} = 2x + 1$, $\frac{\partial N}{\partial x} = 2x + 1$. Exact.

Step 1: Integrate $N$ w.r.t $y$: $\int (x^2 + x) dy = x^2y + xy + f(x)$.

Step 2: Differentiate w.r.t $x$: $2xy + y + f'(x) = M = 2xy + y \implies f'(x) = 0$.

✅ Final Answer: $x^2y + xy = C$

Q8. Solve: $\frac{dy}{dx} = \frac{x+y}{x-y}$

Homogeneous ODE

Homogeneous equation. Let $y = vx$, then $\frac{dy}{dx} = v + x \frac{dv}{dx}$.

Step 1: $v + x \frac{dv}{dx} = \frac{x+vx}{x-vx} = \frac{1+v}{1-v}$.

Step 2: $x \frac{dv}{dx} = \frac{1+v}{1-v} - v = \frac{1+v^2}{1-v} \implies \frac{1-v}{1+v^2} dv = \frac{dx}{x}$.

Step 3: $\int \frac{1}{1+v^2} dv - \int \frac{v}{1+v^2} dv = \ln|x| + C$.

Step 4: $\arctan(v) - \frac{1}{2}\ln(1+v^2) = \ln|x| + C$.

✅ Final Answer: $\arctan(y/x) - \ln\sqrt{x^2+y^2} = C$

Q9. Given $y_1 = x, y_2 = x e^x$, find Wronskian $W(y_1, y_2)$ and test linear independence on $(0, \infty)$.

Wronskian

Step 1: \[ W(y_1, y_2) = \begin{vmatrix} x & x e^x \\ 1 & e^x + x e^x \end{vmatrix} = x^2 e^x \]

Step 2: Since $x^2 e^x \neq 0$ for all $x \in (0, \infty)$, the functions are linearly independent.

✅ Final Answer: $W = x^2 e^x \neq 0$; Functions are Linearly Independent.

Q10. Solve: $\frac{d^2y}{dx^2} - 5\frac{dy}{dx} + 6y = 0, \quad y(0) = 1, \quad y'(0) = 5$.

Higher-Order ODE

Characteristic Eq: $r^2 - 5r + 6 = 0 \implies (r-2)(r-3) = 0$. Roots: $r = 2, 3$.

Step 1: $y = c_1 e^{2x} + c_2 e^{3x}$.

Step 2: $y(0) = c_1 + c_2 = 1$.

Step 3: $y' = 2c_1 e^{2x} + 3c_2 e^{3x} \implies y'(0) = 2c_1 + 3c_2 = 5$.

Step 4: Solving: $c_2 = 3, c_1 = -2$.

✅ Final Answer: $y = 3e^{3x} - 2e^{2x}$

Q11. Solve: $\frac{d^2y}{dx^2} - 4\frac{dy}{dx} + 4y = 0, \quad y(0) = 0, \quad y'(0) = 0$.

Higher-Order ODE

Characteristic Eq: $r^2 - 4r + 4 = 0 \implies (r-2)^2 = 0$. Root: $r = 2$ (Repeated).

Step 1: $y = (c_1 + c_2 x)e^{2x}$.

Step 2: $y(0) = c_1 = 0$.

Step 3: $y' = c_2 e^{2x} + 2(c_1 + c_2 x)e^{2x}$. $y'(0) = c_2 + 2c_1 = 0 \implies c_2 = 0$.

✅ Final Answer: $y = 0$ (Trivial Solution)

Q12. Solve: $\frac{d^2y}{dx^2} + 2\frac{dy}{dx} + 5y = 0, \quad y(0) = 0, \quad y'(0) = 0$.

Higher-Order ODE

Characteristic Eq: $r^2 + 2r + 5 = 0$. Roots: $r = \frac{-2 \pm \sqrt{4-20}}{2} = -1 \pm 2i$.

Step 1: $y = e^{-x}(c_1 \cos 2x + c_2 \sin 2x)$.

Step 2: $y(0) = 0 \implies c_1 = 0$.

Step 3: $y'(0) = 0 \implies c_2 = 0$ (following differentiation and evaluation).

✅ Final Answer: $y = 0$ (Trivial Solution)
