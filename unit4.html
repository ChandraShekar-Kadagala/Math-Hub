Unit 4

Higher Order Differential Equations

The general solution is always \(y = y_c + y_p\). First find the complementary solution, then the particular.

Topic 1
📐 The Wronskian — Linear Independence Test
Definition: \[W(y_1,y_2) = \begin{vmatrix}y_1 & y_2 \\ y_1' & y_2'\end{vmatrix} = y_1 y_2' - y_2 y_1'\]
Result: If \(W \neq 0\) on an interval, then \(y_1\) and \(y_2\) are linearly independent on that interval.
Solved — Q9 of Assignment

Given \(y_1 = x\), \(y_2 = xe^x\). Find \(W(y_1,y_2)\) and test independence on \((0,\infty)\).

  1. \(y_1 = x, \quad y_1' = 1\)
  2. \(y_2 = xe^x, \quad y_2' = e^x + xe^x\)
  3. \(W = x(e^x + xe^x) - (1)(xe^x) = xe^x + x^2e^x - xe^x = x^2e^x\)
  4. On \((0,\infty)\): \(x^2e^x > 0\), so \(W \neq 0\). → Linearly Independent.
Wronskian = \(x^2 e^x \neq 0\). Functions are Linearly Independent.
Topic 2
⚙️ Homogeneous Linear ODEs — Constant Coefficients
Form: \(ay'' + by' + cy = 0\)

Characteristic Equation: \(ar^2 + br + c = 0\)
Solve for \(r\) using the quadratic formula.
RootsGeneral Solution
Real & Distinct: \(r_1 \neq r_2\)\(y = c_1 e^{r_1 x} + c_2 e^{r_2 x}\)
Real & Repeated: \(r_1 = r_2 = r\)\(y = (c_1 + c_2 x)e^{rx}\)
Complex: \(r = \alpha \pm i\beta\)\(y = e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\)
Solved — Q10 of Assignment (Distinct Roots + IVP)

Solve: \(y'' - 5y' + 6y = 0\),   \(y(0)=1,\; y'(0)=5\)

  1. Characteristic equation: \(r^2-5r+6=0 \Rightarrow (r-2)(r-3)=0 \Rightarrow r=2,3\)
  2. General solution: \(y = c_1e^{2x} + c_2e^{3x}\)
  3. Apply \(y(0)=1\): \(c_1+c_2=1\)
  4. \(y' = 2c_1e^{2x}+3c_2e^{3x}\). Apply \(y'(0)=5\): \(2c_1+3c_2=5\)
  5. Solve: \(c_2=3, c_1=-2\)
Answer: \(y = 3e^{3x} - 2e^{2x}\)
Solved — Q11 (Repeated Root + IVP)

Solve: \(y'' - 4y' + 4y = 0\),   \(y(0)=0,\; y'(0)=0\)

  1. Char. eq: \(r^2-4r+4=0 \Rightarrow (r-2)^2=0 \Rightarrow r=2\) (repeated)
  2. General solution: \(y = (c_1 + c_2 x)e^{2x}\)
  3. \(y(0)=0 \Rightarrow c_1=0\).   \(y' = c_2e^{2x}+2c_2xe^{2x}\). \(y'(0)=0 \Rightarrow c_2=0\).
Answer: \(y = 0\) (Trivial Solution — both constants are zero due to zero ICs)
Solved — Q12 (Complex Roots + IVP)

Solve: \(y'' + 2y' + 5y = 0\),   \(y(0)=0,\; y'(0)=0\)

  1. Char. eq: \(r^2+2r+5=0\). Discriminant: \(4-20=-16 < 0\) → complex roots.
  2. \(r = \dfrac{-2\pm\sqrt{-16}}{2} = -1 \pm 2i\). So \(\alpha=-1, \beta=2\).
  3. General solution: \(y = e^{-x}(c_1\cos 2x + c_2\sin 2x)\)
  4. \(y(0)=0 \Rightarrow c_1=0\). Both ICs zero → trivial solution.
Answer: \(y = 0\) (Trivial Solution)
Topic 3
🌀 Euler-Cauchy Differential Equations
Form: \(x^2 y'' + axy' + by = 0\)   (non-constant coefficients, but pattern!)

Substitution: Assume \(y = x^m\). This gives \(m(m-1) + am + b = 0\) (Auxiliary Equation).
RootsGeneral Solution
Real & Distinct: \(m_1 \neq m_2\)\(y = c_1 x^{m_1} + c_2 x^{m_2}\)
Real & Repeated: \(m_1=m_2=m\)\(y = x^m(c_1 + c_2\ln x)\)
Complex: \(m = \alpha \pm i\beta\)\(y = x^\alpha[c_1\cos(\beta\ln x)+c_2\sin(\beta\ln x)]\)
⚠️ Common Mistake: Using \(e^{rx}\) instead of \(x^m\) for Euler-Cauchy. The form is \(y=x^m\), NOT \(y=e^{rx}\)!
Topic 4
🎲 Method of Undetermined Coefficients

Used for non-homogeneous equations \(ay''+by'+cy = g(x)\). First solve the homogeneous part, then guess \(y_p\).

Right-Hand Side \(g(x)\)Guess for \(y_p\)
\(e^{ax}\)\(Ae^{ax}\)
\(\sin bx\) or \(\cos bx\)\(A\cos bx + B\sin bx\)
Polynomial \(x^n\)\(A_0 + A_1x + \cdots + A_nx^n\)
\(xe^{ax}\)\((Ax+B)e^{ax}\)
⚠️ Modification Rule: If your guess is already in the complementary solution \(y_c\), multiply your guess by \(x\). If it's still there, multiply by \(x^2\).
Topic 5
🔬 Variation of Parameters

A general method for \(y'' + Py' + Qy = g(x)\). Works even when undetermined coefficients fails (e.g., \(g(x) = \tan x\)).

If \(y_c = c_1 y_1 + c_2 y_2\), then: \[y_p = u_1 y_1 + u_2 y_2\] where: \[u_1 = -\int\frac{y_2 g(x)}{W}\,dx, \qquad u_2 = \int\frac{y_1 g(x)}{W}\,dx\] and \(W = W(y_1, y_2)\) is the Wronskian.
Topic 6
📉 Reduction of Order

Used when one solution \(y_1\) is already known. Find a second independent solution.

Assume: \(y_2 = v(x) \cdot y_1\)
Substitute into the ODE. The equation reduces to a first-order ODE in \(v'(x)\). Solve for \(v\), then \(y_2\).
